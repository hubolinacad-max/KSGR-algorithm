# KSGR Algorithm

This repository provides the source code for the KSGR algorithm proposed in the manuscript:

**KSGR: An Influential Node Identification Algorithm for Directed Networks Integrating Reverse Reachability and Structural Enhancement Mechanisms**

## Overview

KSGR is an influential node identification algorithm for directed networks. It integrates reverse local reachability, reverse diffusion efficiency, hierarchical asymmetry, directed reverse k-shell, reverse structural gravity, bridging capability, and a structural diversity enhancement mechanism.

The code is provided to support transparency and reproducibility for the submitted manuscript.

## Files

* `KSGR_upload.ipynb`: Jupyter Notebook containing the implementation of the KSGR algorithm.
* `requirements.txt`: Python dependencies required to run the code.
* `LICENSE`: Open-source license for the repository.
* `CITATION.cff`: Citation information for this code repository.

## Requirements

The code was developed in Python and mainly depends on:

* networkx
* numpy

Required Python packages are listed in `requirements.txt`.

## Usage

The implementation is provided in the Jupyter Notebook file `KSGR_upload.ipynb`.

To run the code, users may open the notebook in Jupyter Notebook or JupyterLab and execute the cells sequentially.
